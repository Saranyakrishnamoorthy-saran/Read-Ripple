Read Ripple is a full-stack online bookstore application developed using React.js for the frontend, Java Spring Boot for the backend, and PostgreSQL as the database.
The application supports role-based access for both Admin and Users.

Features:
User

* View new arrivals and browse all books

* Add books to cart and wishlist

* Access blog and contact sections

Admin

* Admin dashboard for book management

* Add, update, delete, and search books by author

* View all books in a structured table format

Tech Stack

* Frontend: React.js, HTML, CSS, JavaScript

*Backend: Java, Spring Boot

* Database: PostgreSQL

Architecture

* RESTful APIs used for frontendâ€“backend communication

* Clean separation of frontend and backend modules
